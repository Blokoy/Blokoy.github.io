ARTS SETUP

Double Castoria, Miyamoto Musashi (berzeker)

This team requires a max limit broken (MLB) Black Grail craft escence which offers an NP damage boost of 80% while loosing 500HP every turn [Demerit]. A Fragment of 2004 Mystic Code can be used to pair for better looping while Chaldea Uniform - Arctic can be used for higher 3rd turn NP damage if desired.
